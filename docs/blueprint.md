# **App Name**: Altek Green VSD Data Base

## Core Features:

- VSD Data Capture: Capture serial numbers, equipment IDs, and other data from VSDs controlling pump motors.
- Equipment Details: Record details about the equipment each VSD is driving, including pump specifications, maintenance schedules, and performance metrics.
- Data Storage: Store all captured data in a structured manner for easy access and reporting in a Cloud SQL Database.
- Reporting: Generate reports summarizing VSD and equipment performance, maintenance history, and potential issues.
- Maintenance Schedule Generation: Generate a maintenance schedule for the equipment on a weekly, monthly, and 3-monthly schedule.
- Task Generation: Generate tasks for the frequency maintenance to be completed.
- Customer Document Generation: Generate a document that can be supplied to the customer for capturing the details of the scheduled maintenance.
- Breakdown History Page: Generate a page for breakdown that is happening of the equipment to be able to have a history profile of such equipment.

## Style Guidelines:

- Primary color: Deep Blue (#2962FF) for a sense of reliability and technical expertise.
- Background color: Light gray (#F0F4F8) to provide a clean, neutral backdrop.
- Accent color: Teal (#40D4BB) to draw attention to interactive elements and key data points.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable style suitable for technical data.
- Use clear, technical icons to represent different data points, equipment types, and VSD parameters.
- Implement a data-dense, information-rich layout that balances readability and accessibility.
- Use subtle animations to highlight changes in data or system status, providing real-time feedback.
